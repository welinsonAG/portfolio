## 🧠 Visão Técnica

Este projeto foi desenvolvido com foco em:

- Estrutura de código limpa e de fácil manutenção
- HTML5 semântico
- CSS3 moderno (Flexbox e Grid)
- Design Responsivo (abordagem Mobile-First)
- JavaScript (ES6+) para manipulação do DOM e interatividade
- Boas práticas de acessibilidade (ARIA labels)
- Otimização de performance
